Greenworks Full Stack Project


[In Progress...]

This project is a full stack project; a website that utilizes the Tesseract Optical Character Recognition (OCR) Engine, the Firebase Realtime Database, Google Analytics, Google Geolocation API, and of course, a website including HTML, CSS, and Javascript. The key feature of this site includes: the Tesseract engine is used to scan the images uploaded by users of a Resin Triangle on their products; it then provides information about their certain product which is pulled from the database. This web application is mobile friendly and responsive with the application of bootstrap components. I am currently building this website with a team of creative professionals to help raise awareness about plastic pollution in our environment.

Plates and Petals, LLC. Website


This website was designed for the event / party planning business of a client, Allyson Forte. Each component of this website is a product of what Forte viewed her website to be like, elegant, fancy, and modern but also simplistic. Ranging from the color schemes, to the fonts, layout, and logo (made by yours truly), this website was exactly what she invisioned. This site was made strictly for marketing purposes for her business, Plates and Petals, and is currently utilized in order to gain more clients. Each photo on the site was edited using softwares like Photoshop, Illustrator, and Lightroom in order to establish a more appealing site and fits the color schemes requested by the client. This website is also mobile friendly and responsive.

GladWags Grooming Website


Gladwags Grooming was a client provided by Monmouth University who was seeking for a simple, appealing website in order to attract more customers or help customers contact them. This grooming service had been around fairly long, but were using an old, basic HTML website to put their information on. The color scheme, design, and layout was a representation of this client's needs. They wanted something old school, but also a touch of modern and elegant styles with interesting ways to display information about their services and contacts. They also wanted to showcase their prized pet groomings, which was done on the home page.
